## 동물병원 이용 후기 사이트
+ Spring MVC 기반
+ Chrome에 최적화 됨
+ 동물병원 이용 후기 아카이빙 사이트 구현<br><br><br>


## 구현된 기능
<br>


+ 일반 사용자

```
(1) 회원가입
   -이메일 본인 인증 
(2) 로그인
(3) 로그아웃
(4) 회원정보 수정

(5) 비밀번호 찾기
   -이메일 본인 인증 
   -임시 비밀번호 이메일 발송

(6) 지역별 병원 리스트
   - 병원 상세 정보 조회(평점순)
   - 태그별 병원 검색(다중 선택 가능)

(7) 병원별 리뷰 게시판(일반 게시물 형식)
   - 페이징
   - 게시글 조회 (첨부파일 조회/다운로드 포함)
   - 게시글 등록 (첨부파일 등록 포함)
   - 게시글 수정 (첨부파일 수정 포함)
   - 게시글 삭제 (첨부파일 삭제 포함)
   - 게시글 검색
   - 댓글 조회
   - 댓글 수정
   - 댓글 삭제
   - 리뷰 추천
   - 리뷰 신고
   - 유저 추천
   - 유저 신고

(8) 병원별 리뷰 게시판(SNS 형식)
   - 게시글 조회 (첨부파일 조회/다운로드 포함(마우스 우클릭))
   - 게시글 등록 (첨부파일 등록 포함)
   - 게시글 수정 (첨부파일 수정 포함)
   - 게시글 삭제 (첨부파일 삭제 포함)

(9) 공지사항 게시판
   - 공지사항 조회
   - index 페이지에서 최근 5건의 공지사항 리스트 확인 가능
```
<br>

+ 관리자
```
(1) 로그인
(2) 로그아웃

(3) 공지사항 게시판
   - 공지사항 조회(첨부파일 조회/다운로드 포함)
   - 공지사항 등록(첨부파일 등록 포함)
   - 공지사항 수정(첨부파일 수정 포함)
   - 공지사항 삭제(첨부파일 삭제 포함)

(4) (탈퇴)회원관리
   - 회원 정보 조회
   - 회원 검색
   - 탈퇴 요청 회원 조회
   - 탈퇴 요청 회원 검색
   - 최종 회원 탈퇴 처리
 
(5) 병원 관리
   - 등록 병원 정보 조회
   - 신규 병원 등록
   - 병원 정보 수정
   - 등록 병원 삭제

(6) 병원 태그(병원 세부 특징:응급실 운영 등) 관리
   - 등록 태그 조회
   - 신규 태그 등록
   - 등록 태그 삭제
```
